Q-RAS Plugin is a preprocessing for geometry preparation in HEC-RAS.

The steps to using Q-RAS Plugin are fairly simple:

  1. Load in QGIS a Digital Elevation Model in grid format in a projected crs
  2. Digitize a stream network vector line from upstream do downstream
  3. Digitize a Cross sections vector line from left overbank to righ toverbank (it’s not necessary to follow a particular order in XS digitizing as the plugin order the XS basing on the river feature)
  4. Designate where to store your new geometry file
  5. press OK
  6. Open HEC-RAS and enjoy
